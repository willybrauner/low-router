# Monorepo template

Create typescript library on demand.

## Features

- pnpm 
- turbo
- tsup
- codesandox ci
- changeset
- github action
- tests vitests
- limit-size

## Install

```shell
pnpm i
```

## Start

Listen build watch

```shell
pnpm run build:watch
```

Link your library as dependency in examples package json. When build:watch is running, it will be updated on change.
